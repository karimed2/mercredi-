-------------------------------------------------------
                Projet de Création de Compte
-------------------------------------------------------

Ce projet permet aux utilisateurs de créer un compte, de se connecter et de valider leurs informations à l'aide de PHP et MySQL. Le système comprend trois parties principales :

1. Page de connexion : Permet à un utilisateur de se connecter avec un identifiant et un mot de passe.
2. Page de création de compte : Permet à un utilisateur de créer un nouveau compte si celui-ci n'existe pas déjà.
3. Base de données MySQL : Utilisée pour stocker les informations des utilisateurs (identifiant et mot de passe).

-------------------------------------------------------
                 Fonctionnalités
-------------------------------------------------------

- **Création de compte** : Lorsqu'un utilisateur crée un compte, ses informations sont validées et enregistrées dans une base de données MySQL.
- **Connexion** : L'utilisateur peut se connecter en vérifiant si l'identifiant et le mot de passe correspondent à ceux enregistrés dans la base de données.
- **Gestion des erreurs** : Le système gère les erreurs, telles que les identifiants déjà existants ou les erreurs d'insertion dans la base de données.

-------------------------------------------------------
                   Prérequis
-------------------------------------------------------

- PHP 7 
- MySQL
- WampServer64
